# OntoGuard Website Videos

This folder contains buyer-facing video assets used by the OntoGuard AI public website.

Required production asset:

- `OntoGuard_Video.mp4` — referenced by `index.html` as `videos/OntoGuard_Video.mp4`.

GitHub Pages is case-sensitive. Do not rename the MP4 unless the HTML references are updated at the same time.

Recommended optional accessibility assets:

- `OntoGuard_Video.vtt` — English captions track.
- `OntoGuard_Video_transcript.txt` — plain-text transcript.
